# algoTrad

> Algorithmic trading experiments and strategies using Python and Jupyter Notebooks.

## Description

AlgoTrad is a collection of Jupyter Notebooks exploring algorithmic trading strategies. It covers data reading, futures trading, options analysis, and trading bot prototyping using Python financial libraries.

## Stack / Tech
- Python
- Jupyter Notebooks
- Pandas / NumPy
- Financial data APIs

## Getting Started

```bash
pip install -r requirements.txt
jupyter notebook
```
