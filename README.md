# botzilla

> An automated cryptocurrency trading bot for the Bittrex exchange.

## Description
Botzilla is a .NET cryptocurrency trading bot designed for the Bittrex exchange. It implements automated trading strategies, monitors market conditions, and executes buy/sell orders based on configurable rules.

## Features
- Automated trading on Bittrex exchange
- Configurable trading strategies
- Market monitoring and order execution
- Trade history and portfolio tracking

## Getting Started
```bash
git clone https://github.com/phmatray/botzilla
cd botzilla
# Configure your Bittrex API keys in appsettings.json
dotnet run
```
